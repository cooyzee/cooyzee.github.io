node-sass 安装失败

使用淘宝镜像：--registry=https://registry.npm.taobao.org

指定sass binary site：SASS_BINARY_SITE=https://npm.taobao.org/mirrors/node-sass/

Windows 下失败了？试试 npm install node-sass --sass-binary-site=；或者修改npm配置文件，参考目录： C:\Program Files\nodejs\node_modules\npm
